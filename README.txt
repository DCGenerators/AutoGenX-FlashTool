AutoGen X USB Flash Tool

1) Plug the AutoGen X controller into USB.
2) Run:
   - Windows: double-click FLASH_WINDOWS.bat
   - Mac: double-click FLASH_MAC.command
3) Wait for "DONE / Flash complete".

If it fails:
- Unplug/replug USB and try again.
- Use a DATA USB cable (not charge-only).
- Send Nick a screenshot of the whole window.
